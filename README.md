# BMSScreen2025
The BMS Info Screen in the front of the car(rewritten in C++)

## Pinouts
The Screen Wiring for the RP2040 - SSD1305 Screen is as follows
GND(23) - GND(1)\
3v3(out)(36) - VCC(2)\
GP14(19) - DC(4)\
GP18(24) - SPI CLK(7)\
GP19(25) - SPI DATA IN(8)\
GP21(27) - CS(15)\
GP15(20) - RESET(16)\
